# Relativistic Sunyaev-Zel'dovich Effect Simulation

## Overview
This repository contains a suite of Python notebooks and data files designed to simulate different order Sunyaev-Zel'dovich (SZ) signals. It applies Internal Linear Combination (ILC) and constrained ILC (cILC) algorithms to separate cosmological components, with the primary goal of extracting the kinematic term from the relativistic SZ effect.

## Features
- **CMB Simulation:** Simulates Cosmic Microwave Background (CMB) temperature maps using the TT power spectrum generated by CAMB.
- **SZ Effect Modeling:** Simulates multi-frequency maps containing thermal SZ (tSZ), kinematic SZ (kSZ), and higher-order relativistic SZ effects.
- **Realistic Noise Modeling:** Utilizes the frequency channels and sensitivities of the CMB-S4 telescope to inject realistic instrumental and sky noise into the simulated maps.
- **Component Separation (ILC & cILC):** Tests Internal Linear Combination and constrained ILC methods on first-order and higher-order SZ effects to analyze noise variance and extract specific signal terms.
- **Cluster Stacking:** Employs cluster stacking techniques to minimize the effect of noise and retrieve the cross-velocity term.
- **Realistic Mass Catalogs:** Integrates CCAT-prime cluster masses and Simons Observatory (SO) baseline configurations to calculate the expected Signal-to-Noise Ratio (SNR) for the extracted signal.

## Repository Contents
- **`SZ_functions.ipynb`**: Core functions and utilities for calculating and simulating various SZ effect components.
- **`Testing ILC and cILC on the simulated maps of higher SZ effects.ipynb`**: Applies component separation algorithms to the simulated multi-frequency maps to isolate the relativistic SZ signal.
- **`Cluster_stacking.ipynb`**: Implements the cluster stacking methodology to suppress noise and enhance the SNR of the targeted SZ signatures.
- **`Realistic Cluster Masses to extract kinetic term from higher orderSZ.ipynb`**: Notebook utilizing realistic cluster mass distributions to evaluate the extraction of the kinetic term from higher-order SZ effects.
- **`CAMB_fiducial_cosmo_scalCls.txt`**: Fiducial cosmology scalar $C_\ell$ (TT power spectrum) used as the input for simulating the primary CMB background.
- **`*.fits` data files**: FITS tables (e.g., `nclusters_SO_base_CCATp_combined...`) containing cluster catalog data and noise models combining SO baseline and CCAT-prime configurations.

## Getting Started

### Prerequisites
To run the Jupyter notebooks in this repository, you will likely need a standard cosmological Python environment. Common dependencies include:
- `numpy`
- `matplotlib`
- `scipy`
- `astropy` (for handling `.fits` files)
- `healpy` (if simulating full-sky or spherical map projections)
- `camb` (if you intend to regenerate power spectra)

### Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/kaushikmastro/Relativistic-SZ-effect-simulation.git
   cd Relativistic-SZ-effect-simulation
   ```
2. Launch Jupyter Notebook or Jupyter Lab:
   ```bash
   jupyter notebook
   ```
3. Start by exploring `SZ_functions.ipynb` to understand the core physics, and then proceed to the ILC testing or stacking notebooks to see the analysis pipeline in action.
